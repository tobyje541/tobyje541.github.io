## 📝 Understanding the technology

This software uses 2D Gaussian functions to map light phase and intensity. Traditional methods often lose detail during diffraction calculations. This implementation treats light as a complex value, which preserves subtle depth and edges in the final output. The Gaussian representation allows for smooth transitions in the hologram, reducing visual noise. You do not need to understand complex math to use these functions, as the software handles the calculations for you.
